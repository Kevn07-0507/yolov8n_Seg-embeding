# 墙面裂缝检测系统 - 项目总结

## 项目概述

这是一个基于YOLOv8的完整墙面裂缝检测和分割系统，包含从数据准备、模型训练、推理预测到部署的全流程解决方案。

### ✅ 完整的工作流程
- 数据集分析和统计

- 单张/批量/实时预测
- 结果可视化和报告生成

### ✅ 多种使用方式
- 命令行脚本
- 交互式菜单
- Web界面（Gradio）
- REST API（可扩展）

### ✅ 高级功能

- 数据增强预览

## 使用流程

### 快速开始流程
```bash
1. python quick_start.py      # 环境检查
2. python analyze_dataset.py  # 数据分析
3. python train_crack_seg.py  # 训练模型
4. python predict_crack_seg.py # 预测测试
5. python web_app.py          # 启动Web界面
```
